# 配置

### book.json配置文件

**gitbook**

`{ "description": "This is such a great book!" }`

**description**

`{ "description": "This is such a great book!" }`

#### styles

这个选项是用来自定义书本的css的。

例子：

```js
{
    "styles": {
        "website": "styles/website.css",
        "ebook": "styles/ebook.css",
        "pdf": "styles/pdf.css",
        "mobi": "styles/mobi.css",
        "epub": "styles/epub.css"
    }
}
```

#### plugins

```js
{ "plugins": ["mathjax"] }
```

书本使用的插件列表被定义在 `book.json` 的配置中。

#### pluginsConfig

```js
{
    "plugins": ["myplugin"],
    "pluginsConfig": {
        "myPlugin": {
            "message": "Hello World"
        }
    }
}
```

#### structure

这个选项是用来覆盖GitBook使用的路径的。

例如你想要使用 `INTRO.md` 代替 `README.md`：

```js
{
    "structure": {
        "readme": "INTRO.md"
    }
}
```

#### variables

```js
{
    "variables": {
        "myTest": "Hello World"
    }
}
```
